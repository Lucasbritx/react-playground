## Hydration
![alt text](image.png)
### https://github.com/reactwg/react-18/discussions/37
### https://www.reddit.com/r/reactjs/comments/18fky3q/what_is_hydration_in_react/
Hydration is React “wiring up” event listeners and state to that existing HTML.
HTML rendered in the server and make it interative in the browser. 
React verify the existing HTML and link with the necessary JS to active components, 
allowing inreactivity.
Initial HTML is loaded fast and interativity is available later(after hydrate)

In client only, hydration is not possible as the dom is generated by JS, there
is no HTML pre rendered to be hydrated.
